Subtítulo, clique para editar


Clique aqui para editar este parágrafo. Para adicionar imagens, clique com o botão direito na imagem desejada e em "copiar." Em seguida, clique na área desejada e use CTRL + V para adicionar imagem, na mesma forma como se estivesse colando texto.

Você também pode clicar no botão "imagem" e inserir a URL (botão direito > copiar endereço da imagem) para alterar essa primeira imagem.

Para remover imagens, selecione com o mouse e utilize backspace.

Todos os componentes podem ser selecionados, copiados e colados, agilizando o processo de construção da página.

Formatação de texto


Para formatar o texto, selecione-o com o mouse e, enquanto estiver selecionado, utilize os botões acima para formatar. Exemplos:

Essa é uma palavra com negrito.
Essa é uma palavra com itálico.
Essa é uma palavra com underline.
Esse é um link que redireciona para a página desejada
